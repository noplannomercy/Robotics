# 오라클 딕셔너리 — 오토론 승인 도메인

## TBL_CUSTOMER_MASTER (고객 마스터)

| 컬럼 | 타입 | NULL | 비고 |
|---|---|---|---|
| CUSTOMER_ID | NUMBER(18) | NOT NULL | PK |
| CUSTOMER_NAME | VARCHAR2(100) | NOT NULL | 고객명 |
| RESIDENT_NUMBER | VARCHAR2(13) | NOT NULL | 주민등록번호 (암호화) |
| PHONE | VARCHAR2(20) | NULL | 휴대폰 번호 |
| ANNUAL_INCOME | NUMBER(15) | NULL | 연소득 (원) |
| STATUS | VARCHAR2(10) | NOT NULL | 고객 상태 (ACTIVE / INACTIVE / SUSPENDED) |
| REGISTER_DATE | DATE | NOT NULL | 등록일 |

- PK: `PK_CUSTOMER_MASTER (CUSTOMER_ID)`
- INDEX: `IX_CUSTOMER_RESIDENT (RESIDENT_NUMBER)`

---

## TBL_LOAN_APPLICATION (오토론 신청)

| 컬럼 | 타입 | NULL | 비고 |
|---|---|---|---|
| APPLICATION_ID | NUMBER(18) | NOT NULL | PK |
| CUSTOMER_ID | NUMBER(18) | NOT NULL | FK → TBL_CUSTOMER_MASTER |
| VEHICLE_PRICE | NUMBER(15) | NOT NULL | 차량 가격 |
| LOAN_AMOUNT | NUMBER(15) | NOT NULL | 신청 대출금액 |
| LOAN_PERIOD | NUMBER(3) | NOT NULL | 대출 기간 (개월) |
| STATUS | VARCHAR2(20) | NOT NULL | RECEIVED / APPROVED / REJECTED |
| REJECT_REASON | VARCHAR2(50) | NULL | 거절 사유 코드 |
| APPLY_DATE | DATE | NOT NULL | 신청일 |
| APPROVE_DATE | DATE | NULL | 승인일 |

- PK: `PK_LOAN_APPLICATION (APPLICATION_ID)`
- FK: `FK_LOAN_APP_CUSTOMER (CUSTOMER_ID) → TBL_CUSTOMER_MASTER`
- SEQUENCE: `SEQ_LOAN_APPLICATION`

## TBL_CREDIT_SCORE (신용점수 이력)

| 컬럼 | 타입 | NULL | 비고 |
|---|---|---|---|
| SCORE_ID | NUMBER(18) | NOT NULL | PK |
| CUSTOMER_ID | NUMBER(18) | NOT NULL | FK → TBL_CUSTOMER_MASTER |
| CREDIT_SCORE | NUMBER(4) | NOT NULL | 신용점수 (0~1000) |
| EVAL_DATE | DATE | NOT NULL | 평가일 |
| BUREAU_SOURCE | VARCHAR2(20) | NOT NULL | 평가원 (KCB / NICE) |

- PK: `PK_CREDIT_SCORE (SCORE_ID)`
- FK: `FK_CREDIT_CUSTOMER (CUSTOMER_ID) → TBL_CUSTOMER_MASTER`
- INDEX: `IX_CREDIT_CUST_DATE (CUSTOMER_ID, EVAL_DATE DESC)`

## TBL_NOTIFICATION_LOG (알림 발송 로그)

| 컬럼 | 타입 | NULL | 비고 |
|---|---|---|---|
| LOG_ID | NUMBER(18) | NOT NULL | PK |
| APPLICATION_ID | NUMBER(18) | NOT NULL | FK → TBL_LOAN_APPLICATION |
| NOTIFY_TYPE | VARCHAR2(20) | NOT NULL | APPROVED / REJECT_CREDIT / REJECT_LIMIT |
| PHONE | VARCHAR2(20) | NOT NULL | 발송 번호 |

| SEND_DATE | DATE | NOT NULL | 발송일시 |

- PK: `PK_NOTIFICATION_LOG (LOG_ID)`
- FK: `FK_NOTIFY_APPL (APPLICATION_ID) → TBL_LOAN_APPLICATION`
- SEQUENCE: `SEQ_NOTIFICATION_LOG`
