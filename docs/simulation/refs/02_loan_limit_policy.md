# 대출 한도 산정 정책

## 적용 범위
신용평가를 통과한 오토론 신청 건에 대해 적용 가능한 대출 한도를 산정한다. 신청 금액이 산정 한도를 초과하면 한도 초과 거절 처리된다.

## 기본 한도 공식

기본 한도는 신청 고객의 연소득을 기준으로 산정한다.

- **기본 한도** = 연소득 × 5

연소득은 고객 마스터에 등록된 최신 연소득을 사용한다. 연소득 정보가 누락된 경우 한도 산정이 불가능하므로 별도 검토 대상으로 분류된다.

## 신용점수 가중치

기본 한도에 신용점수에 따른 가중치를 곱하여 최종 한도를 산정한다.

- 신용점수 **800점 이상**: 가중치 1.5
- 신용점수 **700~799점**: 가중치 1.2
- 신용점수 **600~699점**: 가중치 1.0
- 신용점수 **600점 미만**: 가중치 0.7 (단, 600점 미만은 신용평가 단계에서 사전 거절되므로 실제 적용 사례는 없음)

**최종 한도** = 기본 한도 × 가중치

## LTV 제한

신청 단계에서 차량 가격 대비 대출 금액 비율(LTV)이 90%를 초과하는 경우 신청 자체가 반려된다. 이는 한도 산정 이전의 사전 검증 기준이다.

- LTV = 신청 대출금액 ÷ 차량 가격
- LTV 90% 초과 시 거절 사유 코드 `LTV_EXCEEDED`로 즉시 반려

## 한도 초과 처리

산정된 최종 한도보다 신청 금액이 큰 경우 한도 초과 거절 처리된다. 거절 사유 코드는 `LIMIT_EXCEEDED`이며, 신청자에게 한도 초과 알림이 자동 발송된다.

## 대출 기간 제한

오토론의 최대 대출 기간은 60개월이다. 신청 기간이 60개월을 초과하는 경우 신청 단계에서 거절 사유 코드 `PERIOD_EXCEEDED`로 즉시 반려된다.
