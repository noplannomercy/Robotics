# 신청자 알림 발송 정책

## 적용 범위
오토론 신청 처리 결과를 신청자에게 SMS로 자동 발송한다. 알림 발송은 승인 처리 또는 거절 처리가 확정된 직후 트리거된다.

## 알림 유형

알림은 처리 결과에 따라 세 가지 유형으로 분류된다.

- **승인 알림** (NOTIFY_TYPE = `APPROVED`): 최종 승인 처리 시 발송. 메시지: "오토론 승인이 완료되었습니다."
- **신용평가 거절 알림** (NOTIFY_TYPE = `REJECT_CREDIT`): 신용평가 단계에서 거절 시 발송. 메시지: "신용평가 결과 승인이 어렵습니다."
- **한도 초과 거절 알림** (NOTIFY_TYPE = `REJECT_LIMIT`): 한도 산정 단계에서 거절 시 발송. 메시지: "신청 금액이 한도를 초과합니다."

## 발송 채널

발송은 SMS 단일 채널이며, 고객 마스터에 등록된 휴대폰 번호로 발송한다. 휴대폰 번호 미등록 고객은 알림 대상에서 제외되며 별도 오류 처리하지 않는다.

## 발송 로그 기록

모든 발송 시도는 알림 발송 로그에 기록된다. 로그에는 신청 ID, 알림 유형, 발송 번호, 발송 메시지, 발송일시가 저장된다. 발송 실패 시에도 로그는 기록되며, 발송 자체는 비동기로 처리되어 신청 처리 흐름을 차단하지 않는다.

## 신청 단계별 사전 거절은 알림 미발송

LTV 초과(`LTV_EXCEEDED`), 대출 기간 초과(`PERIOD_EXCEEDED`), 비활성 고객(`CUSTOMER_INACTIVE`) 등 신청 접수 단계에서 즉시 반려되는 케이스는 알림 발송 대상이 아니다. 이는 신청서가 정식으로 접수되기 이전 단계의 검증 실패이므로 SMS 발송 비용을 발생시키지 않는다는 정책에 따른다.
